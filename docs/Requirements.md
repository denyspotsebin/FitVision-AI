## 1. Опис предметної області
**FitVision-AI** — це система на базі штучного інтелекту, яка дозволяє користувачам завантажувати свої фотографії, вводити цільові фізичні параметри (вага, BMI, відсоток жиру) та отримувати згенероване AI-зображення очікуваної трансформації тіла.

## 2. Функціональні вимоги

| ID | Функціональна вимога | Пріоритет |
| :--- | :--- | :--- |
| **FR-01** | Система повинна дозволяти користувачу зареєструватися або увійти в акаунт. | Must |
| **FR-02** | Зареєстрований користувач повинен мати можливість завантажити своє фото. | Must |
| **FR-03** | Система повинна дозволяти введення цільових параметрів: бажаної ваги, BMI, відсотка жиру та м’язової маси. | Must |
| **FR-04** | Система повинна генерувати AI-зображення очікуваної трансформації на основі фото та заданих цілей. | Must |
| **FR-05** | Система повинна зберігати результати створених трансформацій в профілі користувача. | Must |
| **FR-06** | Користувач повинен мати можливість переглядати історію своїх попередніх трансформацій. | Should |
| **FR-07** | Система повинна відправляти користувачу повідомлення про готовність результату, оновлення акаунта або нагадування. | Should |
| **FR-08** | Адміністратор повинен мати можливість керувати користувачами, контентом та підтримкою. | Must |
| **FR-09** | Система повинна обмежувати доступ для неповнолітніх осіб. | Must |
